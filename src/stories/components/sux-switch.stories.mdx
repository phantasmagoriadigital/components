# Switch

Switches allow users to turn an individual option on or off. They are usually used to activate or deactivate a specific setting.

export const Defaultswitch = args => {
  return html`
    <div style="padding: 10%; display: flex; justify-content: center; gap:16px;">
      <sux-switch .label="${args.label}" .checked="${args.checked}" .disabled="${args.disabled}"> switch label </sux-switch>
    </div>
  `;
};

<Canvas>
  <Story name="Playground">{Defaultswitch.bind()}</Story>
</Canvas>

## Variants

### Selection

Switches can either be selected or not selected.

<!-- Tag with Info Variant -->

### disabled
A switch in a disabled state shows that a selection exists, but is not available in that circumstance. This can be used to maintain layout continuity and communicate that an action may become available later.

<!-- Tag with Info Variant -->

export const SwitchDisabled = args => {

    </div>
  `;
};
